# TEAM NOVA RANKING

TEAM NOVA RANKING은 팀노바 오픈 카페(<https://cafe.naver.com/teamnovaopen>)에 올라온 작품들을 가져오고<br> 각 단계별 순위를 보여줍니다.<br><br>

* 기초 JAVA
* 기초 Android
* 기초 PHP
* 응용1단계
* 응용2단계
 
랭킹 선정 기준은 댓글 수 + 조회 수 + (5 * 좋아요 수)입니다.<br>
